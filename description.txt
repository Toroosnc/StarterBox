Starter Box
Get a running start in Terraria! When you enter the world for the first time, you'll automatically receive a Starter Box — right-click it to open.
What's inside:
-Starter weapons for all four classes
-Life Crystal & Mana Crystal
-Gold Coins
-Lesser Healing Potion

Magic Storage support:
If you also have the Magic Storage mod installed, the box will automatically include Storage Components, a Storage Heart, and a Storage Crafting Interface — letting you set up a storage system right from the start of the game.
Perfect for players who don't want to grind the early game again, or who just want to jump straight into exploring and fighting bosses without hunting down basic gear first.
