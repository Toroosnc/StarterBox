# 📦 Starter Box

A [tModLoader](https://github.com/tModLoader/tModLoader) mod for Terraria that gives you a running start! When you enter a world for the first time, you automatically receive a **Starter Box** — right-click it to open.

![tModLoader](https://img.shields.io/badge/tModLoader-1.4.4-blue)
![Terraria](https://img.shields.io/badge/Terraria-1.4.4.9-green)

## Features

- Automatically given to every character the first time they enter a world
- Starter weapons for **all four classes**:
- Life Crystal & Mana Crystal
- Gold Coins
- Lesser Healing Potion
- **Magic Storage integration** — if [Magic Storage](https://steamcommunity.com/sharedfiles/filedetails/?id=1276340573) is installed, the box also includes Storage Components, a Storage Heart, and a Storage Crafting Interface

## Installation

### Steam Workshop
Subscribe to the mod on the Steam Workshop, then enable it from the tModLoader Mods menu in-game.

## Mod Compatibility

- **Magic Storage** — detected automatically at load; no hard dependency required. If not installed, the box simply skips those items.

## Known Issues / Notes

- The Starter Box is granted once per character and tracked via player save data, so it won't duplicate if you re-enter the world.
